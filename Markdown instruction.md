# Инструкция для работы с Markdown

## Выделение текста

Чтобы сделать текст курсивом, необходимо обрамить его звездочками (*). 
Например, *вот так*.

Чтобы выделить текст полужирным, необходимо обрамить его двойными звездочкми (**). 
Например, **вот так**
